üí° How Stacks Power Browser History

  • Visit a page: Current page pushed to back stack
  • Back button: Pop from back stack, push current to forward stack
  • Forward button: Pop from forward stack, push current to back stack
  • Visit new page while forward stack exists: Forward stack cleared!
